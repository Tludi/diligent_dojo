== Welcome Diligent Dojo

This is a simple todo Rails 3 app showcasing use of various resources

1.) BDD testing with Rspec and Cucumber
2.) Authentication with Devise gem
3.) Use of twitter bootstrap
4.) Jquery

Todo app lets you make a simple list of todo items then lets you prioritize the top 5.
